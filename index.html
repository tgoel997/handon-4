What is css?

  1. CSS stands for cascading style sheets.
  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  3. CSS is not neccesary CSS saves a lots of work. it can control the layout of multiple web page all at once
  4. External style sheet are stored in CSS files

CSS can be added to HTML documents in 3 ways

  • Inline- by using the style attribute inside HTML document1
  • Internal- by using the style sheet in the head section
  • External- by using link elements to link to an external CSS file.
  • The most common way to add CSS, is to keep the style in external CSS file. However,in this tutorial we will use inline and internal style, because it is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID. The difference is that CSS class applies to style a multiple elements. ID, on the other hand, applies a styele to one unique element. ID is also special in that you can use a special URL to link directly to an element and its use by javascript.


In CSS, selectors are used to target a specific element or range of elements on a web page. once an element has been targeted, a style or set of styles can be applied to the element.


there is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
